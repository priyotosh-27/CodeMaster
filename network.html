OSI & TCP/IP Models

Layered models separate concerns: application, transport, network, link, and physical. We'll map protocols to layers and explain encapsulation/decapsulation at boundaries.

Link Layer & Ethernet

Ethernet framing, MAC addresses, ARP, switching, VLANs. We'll show frame formats and switching tables, and how learning switches avoid broadcasts.

IP, Addressing & Subnetting

IPv4/IPv6 addressing, CIDR notation, subnet masks, NAT, and address allocation. We'll include worked examples of subnet math and prefix aggregation.

Transport Layer — TCP & UDP

TCP handshake, state machine, retransmission, flow control, congestion control (AIMD, slow start, fast recovery). UDP for connectionless protocols and QUIC as modern transport over UDP.

Routing Protocols

Distance-vector (RIP) vs link-state (OSPF) and path-vector (BGP). We'll outline route convergence, metrics, and BGP policy controls used by ISPs.

Application Layer & Security

HTTP/2, HTTP/3, TLS handshake, certificates, DNS resolution and security extensions (DNSSEC). We'll discuss DDoS mitigation, rate limiting, and secure deployment patterns.

Exercises

  1. Calculate subnets for given address space and host requirements.
  2. Trace a TCP connection's packet sequence with handshake, data, and teardown.
  3. Compare link-state and distance-vector in a small topology for convergence time.
